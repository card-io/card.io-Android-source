card.io for Android example
===========================

This sample shows how to integrate card.io in Android.

To give it a try:

## With Gradle (modern)

1. Clone this repo.
2. `$ cd SampleApp`
3. Plug in your device
4. `$ ./gradlew installDebug`
5. Run the demo app!


That's it!!
